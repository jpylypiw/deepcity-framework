## Getting Started:  

### Prerequisites:

- Download **Node.js** with **npm** 

### Clone GitHub Repository:

```bash
git clone https://github.com/Eukalyptus-Dreizack/deepcity-framework.git
```  

### Dependencies:

1. Install **pnpm**
```bash
npm i -g pnpm 
```
2. Install **all dependencies** with **pnpm**
```bash
pnpm i
```
